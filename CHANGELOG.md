## 版本說明

本項目基於 MeterSphere 進行開發，提供 Java 和 Python 兩種後端實現。

- **Java 版本**: 位於 `backend/` 目錄（原版 MeterSphere）
- **Python 版本**: 位於 `backend_python/` 目錄（推薦使用）
