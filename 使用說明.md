# MeterSphere Python Backend - 使用說明

## 🚀 快速開始

### 從項目根目錄運行

```bash
# 啟動所有服務（包括 MySQL、Redis、Kafka、MinIO）
./run.sh

# 停止所有服務
./stop.sh

# 重啟所有服務
./restart.sh

# 查看日誌
./logs.sh          # 查看所有服務日誌
./logs.sh backend  # 查看後端服務日誌
./logs.sh mysql    # 查看 MySQL 日誌
```

## 📋 腳本說明

### run.sh
- 檢查 Docker 環境
- 自動創建 .env 文件（如果不存在）
- 構建並啟動所有服務
- 初始化數據庫
- 驗證服務健康狀態

### stop.sh
- 停止所有 Docker 服務
- 保留數據卷（數據不會丟失）

### restart.sh
- 重啟所有服務
- 驗證服務健康狀態

### logs.sh
- 查看服務日誌
- 支持查看特定服務日誌：`./logs.sh <服務名>`

## 🌐 訪問地址

啟動成功後：

- **API 健康檢查**: http://localhost:8081/api/v1/health
- **Swagger 文檔**: http://localhost:8081/api/docs
- **ReDoc 文檔**: http://localhost:8081/api/redoc
